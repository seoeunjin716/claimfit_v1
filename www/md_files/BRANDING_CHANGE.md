# 모두의 사정 - 브랜딩 변경 전략 문서

## 📋 변경 개요

### 변경 전
- **앱 이름**: 사정의 고수
- **슬로건**: 한 명의 고수가 당신의 보상을 책임집니다

### 변경 후
- **앱 이름**: 모두의 사정 - Claimfit 손해사정
- **슬로건**: 모두의 사정을 말해주세요. 각각 사정에 맞는 손해사정 서비스를 제공합니다.

1. **브랜드 포지셔닝 변경**: "한 명의 고수" 중심 → "모든 사용자의 다양한 사정" 중심
2. **서비스 범위 확장**: 개인 전문가 중심 → 다양한 사정 유형 대응
3. **브랜드 통일성**: "모두의 사정 - Claimfit 손해사정"으로 서비스명 명확화

## ⚠️ 주의사항

### 1. **슬로건 길이 고려**
새 슬로건이 기존보다 길어서 UI 레이아웃에 영향을 줄 수 있습니다:
- **스플래시 화면**: 2줄로 분리 또는 폰트 크기 조정 필요
- **로그인 화면**: 레이아웃 여백 확인
- **홈 화면**: 카드 내부 레이아웃 확인

### 2. **브랜드 일관성**
- 모든 화면에서 동일한 앱 이름과 슬로건 사용
- 앱 이름: "모두의 사정 - Claimfit 손해사정"
- 슬로건: "모두의 사정을 말해주세요. 각각 사정에 맞는 손해사정 서비스를 제공합니다."

### 3. **검색 및 교체 시 주의**
- "사정의 고수"가 코드 주석이나 변수명에 포함되어 있을 수 있음
- 실제 사용자에게 보이는 텍스트만 변경
- 코드 로직에 영향을 주는 부분은 신중하게 검토

### 4. **테스트 체크리스트**
변경 후 다음 항목 확인:
- [ ] 스플래시 화면 텍스트 표시 확인
