# 사정의 고수 - 판례 · 법률 자문 탭 콘텐츠 가이드 (2026년 2월 기준)

**탭 이름** : 판례 · 법률 자문  
**아이콘 추천** : 책장 / 저울 / 법전 (법적 신뢰감 강조)  
**목적** :  
- 객관적·공신력 있는 판례를 통해 사용자 신뢰 구축  
- 손해사정 실무 쟁점(증액·거절·과실비율 등)을 법적 근거로 설명  
- 고수의 전문 해설 + 쉬운 요약으로 접근성 높임  
- 자연스러운 상담·의뢰 유도 (판례 보기 → “내 케이스도?” → 상담 연결)

**디자인 원칙**  
- Mobile First, 클린·미니멀·모던 스타일 (Primary #3B82F6 포인트)  
- 카드 기반 리스트 (rounded-lg, subtle shadow, white 배경)  
- 비회원 공개 : 판례 리스트 + 요약 / 상세 해설·버튼 클릭 시 로그인 유도 팝업  
- Disclaimer (모든 카드 하단) :  
  “본 내용은 참고용이며 법적 자문이 아닙니다. 실제 적용은 고수 상담을 권장합니다.”

## 1. 탭 전체 구조

### 상단 필터링 섹션
- 사고 유형 드롭다운 : 교통사고 / 실손의료 / 배상책임·재물 / 후유장해 / 진단·수술비 / 기타  
- 쟁점 키워드 칩 : 과실비율 / 고지의무 위반 / 보험금 거절 / 증액 인정 / 누수·화재 등  
- 연도 필터 : 최근 5년 / 전체 (기본: 최근 5년)  
- 검색 바 (optional) : “교통사고 증액” 등 키워드 검색

### 메인 콘텐츠 : 판례 카드 리스트 (무한 스크롤)
- 카드 구성 요소  
  - 상단 뱃지 : 사고 유형 (색상 구분: 교통사고-빨강, 실손-파랑 등)  
  - 제목 : “대법원 2024다288861 [구상금] – 조치의무 위반 시 청구 가능”  
  - 키 포인트 (bold) : 1~2줄 핵심 요약  
  - 고수 해설 : 3~5줄 쉬운 설명 (전문 용어 피함)  
  - 하단 :  
    - “판례 원문 보기” 버튼 (웹뷰: 대법원·casenote.kr 링크)  
    - “내 케이스와 비슷해?” 버튼 → 상담 탭 이동 + AI 챗봇에 판례 ID 전달  
    - 고수 서명 : “고수 직접 검토: [고수 이름]”

### 하단 고정 섹션
- **법률 자문 가이드** (아코디언 또는 별도 카드)  
  - 자주 묻는 질문 Q&A (고수가 작성)  
    예: “고지의무 위반 시 보험금 거절되나요?” → 판례 기반 답변  
  - 약관 해석 팁 : 실손보험 고지의무, 자동차보험 과실비율 기준 등  
- **판례 기반 체크리스트**  
  - “이 판례처럼 고지의무 문제 있나요?” → 기존 체크리스트로 연결  
  - 입력 폼 → 항목별 가이드 + “상담 받기” 버튼  
- **큰 CTA 버튼** : “지금 고수와 상담 시작하기” (Primary 색상, 홈·상담 탭 유도)

## 3. 콘텐츠 운영 & 업데이트 계획
- **초기 목표**: 10~15건 (교통 5, 실손 5, 배상 3, 기타 2)  
- **업데이트 주기**: 매월 1~2건 신규 추가  
  - 소스: 대법원 판례속보 (scourt.go.kr), casenote.kr, 국가법령정보센터  
  - 고수 프로필 탭에서 직접 업로드·편집 가능  
- **콘텐츠 제작 팁**  
  - 제목: 판례 번호 + 쟁점 키워드 포함  
  - 해설: 전문 용어 피하고, “증액 포인트”·“거절 극복 팁” 강조  
  - 길이: 요약 100~150자, 해설 200~300자  
  - 이미지: 판례 원문 스크린샷 (익명화) 또는 아이콘 (optional)

## 4. 사용자 행동 유도 플로우
1. 판례 카드 보기 → 키 포인트 + 해설 읽음  
2. “내 케이스와 비슷해?” 버튼 클릭 → 로그인 유도 (비회원 시)  
3. 상담 탭 이동 → AI 챗봇에 “이 판례(2024다288861)처럼 증액 가능할까요?” 자동 입력  
4. 고수 1:1 채팅 연결 → 실제 의뢰 전환
