# "고수" → "손해사정사" 용어 변경 전략 문서

## 📋 변경 개요

### 변경 배경
"모두의 사정" 브랜딩에 맞춰 더 공식적이고 전문적인 용어로 변경하여 신뢰도를 높이고 법적 명확성을 강화합니다.

### 변경 전
- **용어**: "고수" (비공식적, 캐주얼한 느낌)
- **사용 예시**: "고수와 상담", "고수가 확인", "고수 직접 검토"

### 변경 후
- **용어**: "손해사정사" (공식적, 전문적)
- **사용 예시**: "손해사정사와 상담", "손해사정사가 확인", "손해사정사 직접 검토"

---

## 🎯 변경 목적

1. **브랜드 일관성**: "모두의 사정" 브랜딩에 맞는 전문적 톤 유지
2. **법적 명확성**: 공식적인 직업명 사용으로 신뢰도 향상
3. **전문성 강조**: 비공식적 용어 대신 공식 직업명 사용
4. **사용자 경험**: 더 신뢰할 수 있는 전문 서비스 느낌 제공
