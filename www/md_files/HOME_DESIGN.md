# 사정의 고수 - 홈 화면 디자인 최종 조정 전략  
(성공 사례 제거 & 색상 과도함 완전 정리)

**작성일** : 2026년 2월  
**작성자** : EUNJIN (with Grok)  
**목적** :  
- “판례 · 법률 자문” 탭으로 전환하면서 성공 사례 섹션이 중복·필요 없어짐 → 완전 제거  
- 기존 디자인 잔재(과도한 블루·그린 색상, 유치한 느낌) 완전히 정리  
- 홈 화면을 더 세련되고 집중도 높은 첫 인상으로 재구성  
- 신뢰감·전문성·깔끔함을 최우선으로

## 1. 핵심 변경 방향 요약

- **성공 사례 섹션 완전 삭제**  
  - 이유 : 판례 탭으로 대체 완료, 중복 콘텐츠 제거 → 화면이 훨씬 여유로워짐  
  - 효과 : 스크롤 부담 ↓, 고수 브랜딩에 더 집중 가능  

- **색상 과도함 정리**  
  - 기존 문제 : 밝은 블루(#3B82F6)·그린(#22C55E/#10B981)이 과하게 사용되어 시각적 산만함  
  - 목표 : Primary 블루는 버튼·포인트에만 제한 사용, 그린은 뱃지·숫자에만 미니멀하게  
  - 새 팔레트 :  
    - Primary : #2563EB (조금 더 깊고 차분한 블루)  
    - Accent Green : #16A34A (소프트·신뢰감 있는 그린, 뱃지·증액률에만)  
    - Neutral : #F9FAFB (배경), #1F2937 (텍스트), #6B7280 (서브 텍스트)  
    - Success : #059669 (매우 제한적 사용)  

- **전체 느낌** :  
  - 모던·프로페셔널·신뢰감 (참조: Lemonade 앱, Oscar Health, Swiss Re 스타일)  
  - 여백 넉넉, 텍스트 중앙 정렬, 미니멀 아이콘, subtle shadow만  

## 2. 새 홈 화면 구성 (스크롤 가능, 여유로운 레이아웃)

### 상단 40% 영역 – 고수 브랜딩 집중 (첫 인상 핵심)
- 배경 : #FFFFFF 또는 #F9FAFB (깔끔하게)  
- 중앙 정렬 프로필 카드 (rounded-xl, elevation 2, padding 32px)  
  - 원형 프로필 사진 (96px, subtle border)  
  - 이름 : “김고수” (Pretendard Bold 28px, center)  
  - 뱃지 : “직접 수임 · 개인 운영” (소프트 그레이 배경, 작은 텍스트)  
  - 주요 지표 (가로 2열 또는 중앙 정렬)  
    - “평균 증액률 180%” (Accent Green #16A34A, Bold 24px)  
    - “성공 케이스 247건” (Neutral Gray, 18px)  
  - 슬로건 : “한 명의 고수가 당신의 보상을 책임집니다” (Subtitle 16px, center, line-height 1.5)  

- 여백 : 상단 Safe Area + 40px, 하단 32px  

### 중앙 30% 영역 – 즉시 행동 유도 (CTA 중심)
- 큰 Primary 버튼 : “빠른 상담 시작”  
  - #2563EB 배경, white 텍스트, rounded-xl, 높이 56px, full width  
  - 아래 작은 서브 텍스트 : “AI로 먼저 상담받고 고수와 연결” (Gray-600, 14px)  

- 기능 바로가기 카드 그룹 (가로 스크롤 또는 2열 그리드)  
  - 카드 스타일 : rounded-lg, white 배경, subtle shadow, center align  
  - 4개 추천 (아이콘 + 제목 + 짧은 설명)  
    1. 의뢰 신청하기 → 의뢰 탭  
    2. 판례 검색 → 판례·법률 자문 탭  
    3. 증권 제출 확인 → 의뢰 상세 또는 자료 섹션  
    4. AI 상담 먼저 → 상담 탭 (프리미엄 유도 가능)  
  - 아이콘 : 미니멀 라인 아이콘 (Material Icons 또는 Feather Icons 사용)  

### 하단 30% 영역 – 신뢰 지표 & 보조 콘텐츠
- 통계 한 줄 카드 (가로 3열, center align)  
  - “247건” / “180%” / “4.9★” (각각 아이콘 + 숫자 + 라벨)  
  - 배경 : white, rounded-lg, subtle shadow  

- “고수가 책임집니다” 문구 카드  
  - 배경 : 아주 연한 블루 (#EFF6FF), rounded-lg  
  - 텍스트 : “전문가가 직접 상담하고 보상을 책임집니다” (center, 16px)  

- 하단 여백 : 바텀 네비게이션까지 24px 이상  

## 3. 색상·폰트·여백 최종 가이드

### 색상 정리 (과도함 완전 제거)
- Primary Action : #2563EB (버튼, 포커스)  
- Accent Success : #16A34A (증액률, 성공 뱃지 – 과하지 않게)  
- Text Primary : #111827  
- Text Secondary : #4B5563  
- Background : #F9FAFB / #FFFFFF  
- Card Shadow : elevation 1~2 (opacity 0.08)  

### 폰트 & 정렬
- 모든 주요 텍스트 : center align (카드 제목·숫자·슬로건)  
- Title : Pretendard Bold 24–28px  
- Subtitle : Medium 16–18px  
- Body : Regular 14–16px  
- Line-height : 1.5~1.6배 (숨쉬는 느낌)  

### 여백 규칙
- 카드 내부 padding : 24px  
- 섹션 간 margin : 32px  
- 버튼 높이 : 56px (터치 편의)  
- 전체 화면 좌우 margin : 20–24px  

## 4. 개발 우선순위 & 예상 효과

### 우선순위
1. 홈 화면 전체 재구성 (프로필 카드 중앙 정렬 + 성공 사례 제거) – 2주  
2. 색상 팔레트 통일 & 과도한 그린·블루 제거 (전체 앱) – 1주  
3. 기능 바로가기 카드 그룹 구현 – 1주  
총 예상 : 4주

### 예상 효과
- 첫 인상 전문성 점수 (베타 설문) +30~40% 상승  
- 화면 체류 시간 증가 (덜 산만해져서 더 오래 봄)  
- 상담·의뢰 전환율 +20% 이상 목표  
- “유치하다” 피드백 90% 이상 감소
